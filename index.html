Hizmetler

Ben, her doğrultuda web sitesi tasarlayabilir, isteklerinizi göz önünde bulundurarak, aklınızdaki fikirleri gerçeğe dönüştürebilirim. Verdiğim hizmetler aşağıdaki gibi olmakla birlikte sorunuz olması halinde iletişim kurabilirsiniz.

Web Tasarım

Güncel web teknolojilerini göz önünde bulundurarak, sitelerinizi modern olarak tasarlayabilirim.

Web Yazılım

W3C standartlarına uygun olarak güncel web teknolojileriyle, front-end kodlama yapıyorum.

Responsive Tasarım

Tablet, mobil, masaüstü gibi farklı cihazları göz önünde bulundurarak, responsive siteler kodluyorum.

Web Site Bakımı

Web sitenize, isteğiniz doğrultusunda yazılımsal/içeriksel güncelleme yapabilirim.

Yetenekler

HTML/CSS

Olmazsa olmaz.

Web dünyası dendiğinde olmazsa olmaz olan HTML ve CSS ikilisine son derece hakimim. Ne de olsa CSS olmayan bir web sitesi, günümüz dünyasında düşünülemez. Bu doğrultuda web sitelerinizi, buna uygun olarak hazırlayabilir/kodlayabilirim.

JavaScript

Biri etkileşim mi dedi?

İlk izlenim, web siteleri için belki de en önemli faktör. Animasyonları olan, site içerisinde kullanıcıya farklı etkileşimler sunan sitelere illaki denk gelmişsinizdir. Siz de aşık oldunuz değil mi? İşte bu yüzden, JavaScript günümüz web siteleri için olmazsa olmazın bir diğer önemli parçası.

TypeScript

Kimse hata yapmak istemez.

Kimisi için gereksiz gözüksede, büyük çaplı siteler tasarlandığında TypeScript kullanmak avantajlı gözükebilir. Zira JavaScript'in aksine TypeScript kodları derlendiğinden hataları geç olmadan farkedebilir, site geliştirme aşamasını hızlandırabiliriz.

AngularJS

Framework.

Framework kullanılmayan bir JavaScript projesi, sanırım artık düşünülemez. Geliştirme sürecine zaman anlamında ciddi artılar sağlayan Frameworkler geliştiriciler için olmazsa olmazdır diyebiliriz. Bu doğrultuda, ben de projelerimde başta AngularJS olmak üzere farklı farklı Frameworkler kullanmaya özen gösteriyorum.

Adobe Photoshop

Artı denebilir.

Her ne kadar günümüz dünyasında "Front-End" dediğimizde Photoshop, farklı alternatiflerin olması sebebiyle gerekmese de, bazı durumlarda Adobe'nin yazılımlarını bilmek avantaj sağlayabiliyor.
